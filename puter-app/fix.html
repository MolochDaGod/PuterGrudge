📖 System Overview

This document details the complete attribute and stat system for a level 0-20 game. Each attribute point invested provides both flat bonuses(fixed amounts) and percentage bonuses(scaling with base stats).

✅ System Status:Balanced attribute system with proper scaling, diminishing returns, and stat caps to maintain competitive gameplay across all levels.

Core Mechanics

  • Level Range:Characters progress from Level 0 to Level 20
  • Starting Points:20 attribute points at character creation
  • Points Per Level:7 points awarded per level up
  • Maximum Points:20+(20 × 7)=160 total points at level 20
  • Flat Bonuses:Fixed amount added per attribute point (e.g., +26 Health per Strength)
  • Percentage Bonuses:Scales with base stats from level (e.g., +0.8% of base Health per Strength)
  • Diminishing Returns:Reduced efficiency after 25 points in any single attribute
  • Stat Caps:Maximum values for combat stats to maintain balance
Total Stat=(Flat Bonus × Effective Points)+(Base Stat × Percent Bonus × Effective Points)

Effective Points=Full Points (1-25)+Half Points (26-50)+Quarter Points (51+)

🧮 Combat Math & Damage Calculations

Health Calculation

Max Health=Base Health+Attribute Bonuses+Equipment+Traits+Sets

Attribute Bonus=Σ(Flat × Effective Points)+(Base × Σ(Percent × Effective Points))

Example:Level 10, 35 Strength (Effective: 30), 30 Vitality (Effective: 27.5) 

Base Health: 100 (Level 10) Strength: (26×30)+(100 × 0.008×30)=780+24=804 Vitality: (25×27.5)+(100 × 0.005×27.5)=687.5+13.75=701.25 Total: 100+804+701.25=1,
605 HP

Mana Calculation

Same formula applies for Mana resource pool

Example:Level 10, 40 Intellect (Effective: 32.5), 40 Wisdom (Effective: 32.5) 

Base Mana: 50 (Level 10) Intellect: (5×32.5)+(50 × 0.05×32.5)=162.5+81.25=243.75 Wisdom: (20×32.5)+(50 × 0.03×32.5)=650+48.75=698.75 Total: 50+243.75+698.75=992.5 Mana
Diminishing Returns Applied:Effective points calculated as:
• Points 1-25: Full value (1.0×)
• Points 26-50: Half value (0.5×)
• Points 51+: Quarter value (0.25×)

Base Damage=Level Damage+Attribute Bonuses+Equipment+Traits+Sets

Block System

IF Random(0, 1) < Block Chance:
    Damage=Damage × (1 - Block Factor)

Block Chance Cap: 75%
Block Factor Cap: 90% (0.90)

Example:100 damage incoming, 40% block chance, 0.35 block factor

  • 40% chance to trigger block
  • If blocked: 100 × (1 - 0.35)=65 damage taken
  • Modified by attacker's Block Break Factor

Critical Hit System

IF Random(0, 1) < Critical Chance:
    Damage=Damage × Critical Factor

Critical Chance Cap: 75%
Critical Factor Cap: 3.0× (300% damage)

Example:100 base damage, 45% crit chance, 1.8× crit factor

  • 45% chance to trigger critical
  • If critical: 100 × 1.8=180 damage
  • Modified by target's Critical Evasion
At Level 20:With 50 DEX+50 AGI, you can easily reach the 75% crit cap, making crit factor your next priority.

Full Combat Calculation Pipeline

  1. Calculate Base Damage- Level+Attributes+Equipment+Traits
  2. Apply Defense Break- Reduce target defense by attacker's Defense Break Factor
  3. Calculate Mitigation- Apply defense formula (relational damage reduction)
  4. Apply Random Variance- ±25% deviation (if enabled)
  5. Check Block- Roll vs effective block chance (modified by Block Break)
  6. Check Critical- Roll vs effective crit chance (modified by Crit Evasion) if not blocked
  7. Apply Damage- Subtract from target HP
  8. Trigger Effects- Drain, Reflect, Absorb factors activate
Note:Critical hits cannot occur on blocked attacks. Block check happens before critical check.

Drain Mechanics (Lifesteal/Manasteal)

Health Restored=Damage Dealt × Drain Health Factor
Mana Restored=Damage Dealt × Drain Mana Factor

Cap: 50% (0.50)

Reflect Damage

Damage Reflected=Damage Received × Reflect Factor
Cap: 50% (0.50)

Reflect does NOT proc on blocked hits

Absorb Mechanics

Health Regenerated=Damage Received × Absorb Health Factor
Mana Regenerated=Damage Received × Absorb Mana Factor

Cap: 50% (0.50)

Accuracy & Resistance (Debuffs)

Debuff Success=Attacker Accuracy - Target Resistance

Cap: 95% (0.95) - Always 5% minimum chance to resist/land

PlayerAttributes.cs Settings

// Unity Inspector Settings
[Header("Diminishing Returns")]
public bool enableDiminishingReturns = true;
public int diminishingReturnsThreshold = 25;
public float tier1Efficiency = 0.5f;  // 50% for points 26-50
public float tier2Efficiency = 0.25f; // 25% for points 51+

// Calculation Method
public float CalculateEffectivePoints(int actualPoints)
{
    if (!enableDiminishingReturns || actualPoints <= diminishingReturnsThreshold)
        return actualPoints;
    
    float effective = diminishingReturnsThreshold;
    
    if (actualPoints <= 50)
    {
        // Tier 1: Points 26-50
        effective += (actualPoints - diminishingReturnsThreshold) * tier1Efficiency;
    }
    else
    {
        // Tier 1: Points 26-50
        effective += (50 - diminishingReturnsThreshold) * tier1Efficiency;
        // Tier 2: Points 51+
        effective += (actualPoints - 50) * tier2Efficiency;
    }
    
    return effective;
}

Attributes.Combat.cs - Hard Caps

// Example: Critical Chance Cap
public float criticalChance
{
    get
    {
        float totalCrit = 0f;
        
        // Sum all attribute contributions
        foreach (var attr in attributes)
        {
            totalCrit += attr.template.bonusCriticalChanceFlat * effectivePoints;
            totalCrit += attr.template.bonusCriticalChancePercent * baseStats.critChance * effectivePoints;
        }
        
        // Add equipment, traits, buffs, etc.
        totalCrit += equipment.critChance + traits.critChance + buffs.critChance;
        
        // Apply hard cap
        return Mathf.Clamp(totalCrit, 0f, 0.75f); // 75% cap
    }
}
